Create forms in HTML

Today we are learning about forms in HTML document. Forms are essential for collecting user input and can be used for various purposes such as registration, feedback, and more. Below is a simple form example that includes text inputs, radio buttons, checkboxes, and a submit button.

In the beginning, we will create a simple form that collects a user's name, email, and their favorite programming language. The form will also include a submit button to send the data.

Every form in HTML document starts with tag <form> and ends with tag </form>. Inside the form, we can use various input types such as text, email, radio, checkbox, and submit.

In tag <form> we can specify attributes like action and method. The action attribute defines where the form data will be sent when submitted, while the method attribute specifies how the data will be sent (usually "GET" or "POST").

In the form we use tags <input> for text inputs, <input type="email"> for email inputs, <input type="radio"> for radio buttons, and <input type="checkbox"> for checkboxes. Each input can have a name attribute to identify the data when submitted.

Tag input has these attributes: type, name, value, and placeholder. The type attribute specifies the type of input (e.g., text, email, radio, checkbox), the name attribute is used to identify the input data when the form is submitted, the value attribute defines the value that will be sent when the form is submitted, and the placeholder attribute provides a hint to the user about what to enter in the input field.

With tag input we must to use tag <label> to provide a label for each input field. The for attribute in the label should match the id of the corresponding input to ensure accessibility and usability.

We can unite some forms elements with tag <fieldset> to group related inputs together. This is especially useful for radio buttons and checkboxes, where we can use the <legend> tag to provide a title for the group.

Tag input has a different ways to be used, including:
- Text input: <input type="text">
- Email input: <input type="email">
- Password input: <input type="password">
- Radio buttons: <input type="radio">
- Checkboxes: <input type="checkbox">
- Submit button: <input type="submit"> - Color input: <input type="color">
- Date input: <input type="date">
- Datetime-local input: <input type="datetime-local">
- Month input: <input type="month">
- Week input: <input type="week">
- Time input: <input type="time">
- URL input: <input type="url">
- Tel input: <input type="tel">
- Search input: <input type="search">
- Number input: <input type="number">
- Range input: <input type="range">
- File input: <input type="file">

If we add <input type="radio"> we can use it to create a group of options where only one option can be selected at a time. Each radio button should have the same name attribute to group them together, and the value attribute defines the value that will be submitted when that option is selected.

If we add input type="file" we can use it to allow users to upload files. The <input type="file"> tag creates a file input field where users can select files from their device. We can also specify the accept attribute to limit the types of files that can be uploaded. We can use attribute multiple to allow users to select multiple files at once.

Tag textarea is used for multi-line text input. It allows users to enter larger amounts of text, such as comments or messages. The <textarea> tag can have attributes like rows and cols to specify the size of the text area.

In forms we can use tag <button> to create buttons. The <button> tag can be used for various purposes, such as submitting the form or performing a specific action. The button can have text or an icon inside it. We can use button with attribute type="submit" to submit the form data when clicked. Or with attribute type="reset" to reset the form fields to their default values. Also we can use button with attribute type="button" to create a button that performs a specific action without submitting the form.
