Tables

Long time ago all web pages was created use tables tags. Developers made web pages using only table tags.

But in nowadays tables are used only to show in page some tables data.

First table

Look at the tables tags. We must to use obligate tag: table. In this tag we need to create a little table description using tag: caption where we write some table short description. Important! Tag: caption in one table can be only one time.

After that we use tag: tr what means: table row. That is obligate tag in every table.

In tag: tr we use tag: td, that means: table description.

Also we can use tag: th in tag: tr, to write some bold text. It is like a title, bold text.

Every new tables line we need to make using tags: tr and td.

Tables attributes

Tables has only two attributes: colspan and rowspan. It is necessary if we want to combine some horizontal and vertical lines. We use attribute: colspan to combine horizontal lines, and we use attribute: rowspan to combine vertical lines.

Second Semantic table

We was creating simple table. We use standart tags. But in nowadays for screen reader and another developers, who will working with us code, and we, when we will opening this project after couple years, it will be hard to read this code. It will be difficult for us to navigate it.

Look: How we must to crate Semantic wright table.

First: We need to use right Semantic table structure. It means: we mut to use these tags: table, caption, thead, tbody, tfoot. In tag: thead we create tag: th where we put same table items titles. In tags: tbody, tfoot we must create tag: td, where we put tables items description.

Also, don't forget use tables attributes: colspan and rowspan. Remember! They unites us table horizontal and vertical items.
