All about Links

We can use tag: a to create links in our page. Links can be two types.

First type of links is navigation link: that can be navigate to another page, service. Outside from us page.

Second type: Links who are navigate into our page. To use this type of links, we must in tag a use obligatory attribute: href. In this attribute we must use value: "#block_1". If we press on this link, we can navigate in our page to block_1. To make this possible, in first block we must to use attribute: id and in that value we write unique name.

Important! One ID name in one page, can be only one. It can't be repeated.

Also, we must to use in tag: a attribute: target with value: "_blank". We must to do this for the user to stay in us page, but link will be opening in new window.

If we need to add telephone number phone number as a link, in attribute: href, we must in value write: "tel:+37129909120". Don't forget in link write us phone number in link text. Important! In attribute: href and in link phone number must be the same.

Also, we can add e-mail and skype option to us page use tag: a. If we want to add e-mail, in attribute: href, in value we need to write: "mailto:zanezeltina32@inbox.lv" . Don't forget it write in link. Text must be the same.

If we need to add skype nickname, we in attribute href value must to write: "skype:Zany1919". Remember! In attribute href and in link text must be the same. But it is one thing: In link text: We don't write prefixes: tel, mailto, skype.

Now we must to look, how we can create buttons in our page.

We must to use tag: button with obligatory attribute: type. In attribute type value, we must to write some of these values: "submit, reset, button".

We use value: "submit", if we have a form and we want to send our data to server. Then we press button who have: type="submit".

We use value: "reset", if we have a form in HTML page and we want to reset our entered data.

We use value: "button", if it is just a button, where we can click.

Buttons attributes

Here is block_3

We know one of tag: button attribute. It is type. Tag: button has one more attribute. It is: autofocus. It is used, when we want to automatic make a focus on our button. In page, we can see 1px line around the button. it means, our buttons is in focus.

Important! In one page we can use only one time tag: button attribute: autofocus! We can't use it twice in one page.

Where we need to use attribute: autofocus. it are use in situation, when we need to show to user some modal window, where if he want to click close button in in the upper corner of the window, the close button will be in focus. And all what he must to do is: press Enter on his keyboard.

When we will learns CSS we will learn: CSS property: outline: 1px solid blue; In future, when we will learning CSS, we will change this, but now, we don't need to know it.
